# react
> A JavaScript library for building user interfaces

官方的介绍就是这样很简明的介绍，`一个用于构建用户界面的js库`，

三大特色:
1. 声明式渲染
    * 通过jsx，可以简单直观的编写想要的UI
    * 通过props和state可以自动更新视图
    * 声明式渲染使得状态可预测以及更容易debug
2. 组件式
    * 组件是react编写UI的最小单元
    * 页面将由各种组件组合而成
    * 组件可以是自定义、可以是htmlElement
3. 运行环境多样化
    * 可以在客户端渲染
    * 可以在服务端渲染
